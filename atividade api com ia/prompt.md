
#Prompt inicial

Crie uma classe java modelo para representar uma vaga de emprego que será usada no projeto conecta talentos sendo que uma vaga possui nome, área de atuação, e requisito minimo. 

Crie um endpoint que buscará todas as vagas disponíveis. 

Crie um endpoint que insere novas vagas recebendo todos esses campos como obrigatórios. 

Crie um endpoint que remove uma vaga com base no id dela.
